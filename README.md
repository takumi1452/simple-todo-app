# simple-todo-app

HTML / CSS / JavaScript の学習用として作成したシンプルなToDoアプリです。

---

## 概要
- ToDoの追加 / 削除
- 完了・未完了の切り替え
- localStorageによるデータ保存（ブラウザ再読み込み後も保持）

---

## 使用技術
- HTML
- CSS
- JavaScript（フレームワーク不使用）

---

## 動作環境
- OS：Windows
- ブラウザ：Google Chrome（最新版想定）

---

## 目的
- DOM操作の理解
- イベント処理の理解
- 配列による状態管理の理解
- 設計 → 実装 → 改善の流れを経験すること

---

## 生成AIの使用について
本プロジェクトでは、学習補助として生成AI（ChatGPT 5.2）を使用しています。

- HTML / JavaScript：実装のヒント・設計の方向性について助言を受けた
- CSS：コード例および解説の補助
- requirements.md：要件定義書のテンプレート作成および記述の補助

※ 最終的な理解・実装内容の確認は自身で行っています。
